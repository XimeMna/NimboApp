Concepto de POO

Paradigma de programación que organiza el software alrededor de objetos y datos, en lugar de acciones y lógica.

Clase

Plantilla para crear objetos que define propiedades y métodos comunes a todos los objetos de ese tipo.

Objeto

Instancia concreta de una clase que tiene atributos y métodos específicos.

Encapsulación

Principio que consiste en ocultar el estado interno de un objeto y permitir el acceso solo a través de métodos.

Herencia

Mecanismo que permite que una clase herede atributos y métodos de otra clase, formando una jerarquía.

Polimorfismo

Capacidad de objetos de diferentes clases de responder al mismo mensaje con comportamientos específicos.

Abstracción

Proceso de identificar las características esenciales de un objeto y eliminar los detalles irrelevantes.

Método constructor

Método especial de una clase que se llama automáticamente al crear un nuevo objeto y se utiliza para inicializar sus atributos.

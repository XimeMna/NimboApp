Etiqueta: <html>

Define el comienzo de un documento HTML y contiene todo el contenido de la página web.

Etiqueta: <head>

Contiene metadatos y enlaces a recursos externos como hojas de estilo CSS y archivos JavaScript, así como el título de la página.

Etiqueta: <body>

Contiene todo el contenido visible de la página web, incluyendo texto, imágenes, enlaces, formularios y otros elementos.

Etiqueta: <p>

Define un párrafo de texto en HTML, que se muestra como un bloque de texto separado de otros elementos en la página.

Etiqueta: <a>

Define un hipervínculo en HTML, que permite enlazar a otras páginas web, ubicaciones dentro del mismo documento, etc.

Etiqueta: <img>

Define una imagen en HTML, especificando la ruta de la imagen mediante el atributo src y proporcionando un texto mediante el atributo alt.

Etiqueta: <div>

Define un contenedor, que se utiliza para agrupar y estructurar otros elementos en bloques separados.

Etiqueta: <h1>

Define encabezados de diferentes niveles en HTML, que se utilizan para estructurar y jerarquizar el contenido de la página.

Etiqueta: <ul>

Define una lista desordenada en HTML, que se muestra con viñetas o puntos y se utiliza para enumerar elementos sin un orden específico.

Etiqueta: <ol>

Define una lista ordenada en HTML, que se muestra con números o letras y se utiliza para enumerar elementos en un orden específico.
